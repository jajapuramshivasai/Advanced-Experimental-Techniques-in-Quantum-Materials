# Advanced-Experimental-Techniques-in-Quantum-Materials

Course Outlines: Growth & characterization of quantum materials: Vapour deposition techniques, Thin film growth, Layer transfer process, X-ray diffraction and reflectivity measurements, Elemental analysis, Device fabrication process using lithography and evaporation techniques, Electrical, optical and magneto-transport measurements on quantum materials, Magneto-optic Kerr effect (MOKE) magnetometry
